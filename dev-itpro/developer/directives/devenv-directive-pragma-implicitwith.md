# Pragma ImplicitWith

[!INCLUDE[2020_releasewave2](../includes/2020_releasewave2.md)]

The `#pragma implicitwith` directive can enable or disable warnings on usage of implicit `with` statements in AL code.

## Syntax

```
#pragma implicitwith
....
```
