# ISBREAKPOINTHIT Method (Debugger)
Specifies if a breakpoint is hit in a debugging session.  
  
## Syntax  
  
```  
  
Ok := ISBREAKPOINTHIT   
```  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **True** if a breakpoint is hit in a debugging session; otherwise, **false**.
