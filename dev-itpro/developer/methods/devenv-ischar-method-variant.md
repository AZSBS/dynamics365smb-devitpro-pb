# ISCHAR Method (Variant)
Indicates whether an AL variant contains a Char variable.  
  
## Syntax  
  
```  
  
Ok := Variant.ISCHAR  
```  
  
#### Parameters  
 *Variant*  
 Type: Variant  
  
## Property Value/Return Value  
 Type: Boolean  
  
 **True** if the AL variant contains a Char variable; otherwise, **false**.
