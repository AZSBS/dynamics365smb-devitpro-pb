# NEW Method (TestPage)
Sets the current row of the test page to an empty row in a dataset.  
  
## Syntax  
  
```  
TestPage.NEW  
```  
  
#### Parameters  
 *TestPage*  
 Type: TestPage  
  
 The test page that contains the dataset.
