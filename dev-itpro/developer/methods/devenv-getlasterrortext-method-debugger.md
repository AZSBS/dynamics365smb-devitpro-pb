# GETLASTERRORTEXT Method (Debugger)
Gets the last error that occurred in the debugger.  
  
## Syntax  
  
```  
  
String := GETLASTERRORTEXT  
```  
  
## Property Value/Return Value  
 Type: Text  
  
 A text that represents the last error that occurred.
