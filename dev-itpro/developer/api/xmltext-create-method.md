# Create Method
Creates an XmlText node.  
```  
Value := XmlText.Create(Content)  
```  
## Parameters
*Content*    
&emsp;Type: [String](../datatypes/devenv-text-data-type.md)  
A string that contains the value of the new XmlText node.  
  
## Return Value
*Value*  
&emsp;Type: [XmlText](xmltext-class.md)
