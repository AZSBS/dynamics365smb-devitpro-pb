# ExecutionTimeout Property

Sets the maximum time the report will run after which it is automatically terminated. 
  
## Applies To  

- Reports

## Property Value   

A string in the format `hh:mm:ss`.

## Syntax
```
ExecutionTimeout := `10:05:55`;
```
