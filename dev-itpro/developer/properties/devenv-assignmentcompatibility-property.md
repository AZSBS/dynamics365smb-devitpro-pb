# AssignmentCompatibility Property
Sets the enum to be directly compatible and assignable to an enum of the same type.

## Syntax

```
AssignmentCompatibility = true;
```
&copy;
## Example
<!--
```
SalesOrder.DocumentType := PurchaseOrder.DocumentType;
```
