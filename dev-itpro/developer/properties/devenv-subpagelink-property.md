# SubPageLink Property
Sets a link to a Factbox from a page.  
  
## Applies To  
  
- Pages  

## Syntax example
```
area(factboxes)
{
    part(MyControl;Id)
    {
        ApplicationArea = All;
        PartType = Page;
        SubPageLink = "Table ID" = const(36)","DocumentType"" = field("Document Type")","Document No."" = field("No.");
    }
}
    
```
  
## Remarks  
The link is updated when the current record changes.
