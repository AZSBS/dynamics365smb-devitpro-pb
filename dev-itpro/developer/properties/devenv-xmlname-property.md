# XmlName Property
Sets a variable name for the table or the text that is the data source of the current element.

## Applies To  

- XmlPortTextElement 

- XmlPortFieldElement 

- XmlPortTableElement

- XmlPortFieldAttribute

- XmlPortTextAttribute

## XmlName Type
String

## Syntax
```
XmlName = 'CreditNote';
```
