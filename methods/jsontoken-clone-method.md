# Clone Method
Creates a deep-copy of the JsonToken value.

## Syntax
```
Clone :=   JsonToken.Clone()
```

#### Parameters
*JsonToken*  
&emsp;Type: [JsonToken](jsontoken-data-type.md)  
An instance of the [JsonToken](jsontoken-data-type.md) data type.  

#### Return Value
*Clone*  
&emsp;Type: [JsonToken](jsontoken-data-type.md)
