# Create Method
Creates an XmlText node.

## Syntax
```
Value :=   XmlText.Create(Content: String)
```
## Parameters
*Content*  
&emsp;Type: [String](string-data-type.md)  
A string that contains the value of the new XmlText node.  


## Return Value
*Value*  
&emsp;Type: [XmlText](xmltext-data-type.md)
